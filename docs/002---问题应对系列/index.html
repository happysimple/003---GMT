Note

使用-X选项

gmt subplot begin 4x4 -Fs4c/2.4c -M0.5c -SCb -SRl  -R-180/180/-90/90 -JX4c/2.4c -Xr20c

003.书写带有上下标的文字

@+    打开/关闭上标
@-    打开/关闭下标

004.对nc文件执行数学运算

# 两个nc文件相减（A-B）
gmt grdmath A.nc B.nc SUB = 01.nc

# 乘法
gmt grdmath A.nc 0.00001 MUL = 01.nc
